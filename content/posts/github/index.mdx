4. Configurar el SSH
 a.  Abrir el Git Bash

![GitHub Homepage](/gitbash0.png)

 b.	Copiar el texto debajo, con tu direccion de correo GitHub (ssh-keygen –t rsa –b 4096 –C “tu_correo@ejemplo.com”)

 ![GitHub Homepage](/gitbash1.png)

 c.  Esto crea una nueva clave ssh, utilizando el correo electronico proporcionado como etiqueta
 
 ![GitHub Homepage](/gitbash2.png)

 d.	Cuando se solicite , se ingresa un archivo en donde se va a guardar la clave, presionar enter.Esto acepta la ubicacion predeterminada del archivo, confirmar con (y)
 
 ![GitHub Homepage](/gitbash3.png)
 
 e.  Agregar un contraseña segura ejm: idat16, confirmar contraseña

![GitHub Homepage](/gitbash4.png)
![GitHub Homepage](/gitbash5.png)
![GitHub Homepage](/gitbash6.png)

 f.	Por ultimo agregar el siguiente comando (clip < ~/.ssh/id_rsa.pub)

![GitHub Homepage](/gitbash7.png)

 g.	Ir a GitHub y dirigirse a settings

![GitHub Homepage](/sshconfig1.png)

 h.  Dirigirse a SSH and GPG keys y click en New SSH key

![GitHub Homepage](/sshconfig2.png)

 i.	Colocar un titulo y dar (ctrl + v) en el cuadro grande key  . Click Add SSH key

 ![GitHub Homepage](/sshconfig3.png)
 ![GitHub Homepage](/sshconfig4.png)
